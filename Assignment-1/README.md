# Simple Calculator
This is a basic calculator program that evaluates an arithmetic expression passed as a command-line argument.

## Features
- Supports addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`).
- Takes the expression as a single string argument.
- Prints the evaluated result.

## Usage
Compile the program (C example):
```bash
gcc Assignment-1/1.c -o calculator
```
Run with an expression:
```bash
./calculator "10+5*2"
```
Result:
```
Result: 20
```
